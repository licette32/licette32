### Sobre mí / About

Antes de escribir código necesito entender el problema. Me interesa descubrir cómo funciona un sistema antes de decidir cómo implementarlo.

Mi formación combina ciencia de datos, física, tecnología biomédica y sistemas de información. Esa combinación me llevó a disfrutar proyectos donde el análisis, la arquitectura y la implementación forman parte de una misma solución.

*I enjoy designing systems where data, software and decision-making work together—not as separate components, but as one coherent system.*
